# 👻 NetSpecter

> Scan in Silence. Strike with Precision.

**NetSpecter** is an advanced network scanning and reconnaissance tool inspired by Nmap, built for learning, ethical hacking, and deep network analysis. It focuses on speed, stealth, and modular design.

---

## ⚡ Features

- Fast TCP/UDP port scanning  
- SYN (stealth) & connect scans  
- Service detection & banner grabbing  
- Multi-threaded & scalable engine  
- Clean CLI output
